Lesson 1: Numbers to 1000

In Year 2, we work with numbers up to 1000. We learn about place value - how the position of a digit in a number determines its value. For example, in the number 347, the 3 represents 300 (hundreds), the 4 represents 40 (tens), and the 7 represents 7 (ones).

Practice Questions

What is the value of the 5 in the number 257?

The 5 is in the tens place, so it represents 50.

How would you write 'six hundred twenty-four' as a number?

624

What number is 10 more than 388?

398

Addition and Subtraction Strategies

Mental Strategies for Addition and Subtraction

We learn different strategies to solve addition and subtraction problems mentally. These include counting on, counting back, using doubles, near doubles, and bridging to ten.

For example, to solve 17 + 9, we can think: 17 + 10 = 27, then subtract 1 to get 26. This is called making a friendly number.

Practice Your Mental Maths!

50

Try these calculations: 36 + 27, 53 - 18, 45 + 19

Practice Questions

What is 36 + 27? Use the bridging strategy.

36 + 20 = 56, then 56 + 7 = 63

What is 53 - 18? Use the counting back strategy.

53 - 10 = 43, then 43 - 8 = 35

Measurement

We learn to compare and order objects based on length, area, volume, capacity, and mass. We use informal units like paper clips, blocks, or cups to measure.

Practice Questions

If a pencil is 10 paper clips long, and a ruler is 15 paper clips long, which is longer?

The ruler is longer.

If a container holds 5 cups of water, and another holds 3 cups, which has greater capacity?

The container that holds 5 cups has greater capacity.
